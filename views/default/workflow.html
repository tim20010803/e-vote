Tutorial

Create Account

進入首頁後，可點選Get Started開始，或點選右上進行使用者登入

在建立選舉前，須先建立帳號

在按下signup後，系統會寄送驗證信至所填寫的信箱

至郵件中點選驗證連結即完成註冊!

Create Election

註冊完成並登入後，便可看到管理介面

點選Vote、Manage Election、Ended Election即可切換至各類選舉

點選Create Election即可建立新的選舉

最後按下Save and Preview按鈕後，便可儲存選舉資訊，並可決定是否開始選舉

Ballot and Vote

選票預覽樣式。當創建選舉者按下開始選舉後，選舉便會開始

投票者可透過mail中的選舉連結或是管理介面中的投票按鈕進行投票

投票畫面

當投完票後，投票者畫面會顯示選票已被紀錄

在選舉管理介面，也可觀察到選票已投票的記錄

由於選舉尚未結束，每張選票還不能進去驗證

當選舉結束，投票者可透過mail連結或管理介面觀看選舉結果

Ballot Verification

當選舉結束，每個人都能進入選票介面，觀察選票紀錄

點選Verify Ballot，即可對選票結果與簽證進行檢驗
